# List comprehensions
- [[i, j] for i in range(1) for j in range(1)] // [[0, 0], [0, 1], [1, 0], [1, 1]]
- remove(value): xóa giá trị đầu tiên trong list
- del(index): xóa phần tử tại index
- reverse()
- sort()
- append(): push
- pop()
- insert(index, value): thêm phần tử value tại index
